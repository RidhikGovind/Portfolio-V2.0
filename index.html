About Me *a rant*

I have always been in love with creating stuff from an early age. It started with painting & drawing to shattering my favourite toys into hundreds of individual pieces and fixing them back together to make it work.

But as I grew, it became very hard to break stuff (I mean breaking my phone wasn't a feasible idea), until I found out that you could break stuff by coding. The curiousity, the frustration and the happiness of creating stuff from scratch was really what I was looking for. I wanted to explore more - so I learnt how to build web sites and later on web apps.

Since then, I have learned how to design and build web apps, while also learning the following technologies:

  • HTML, CSS and JavaScript (The Holy Trifecta)

  • React.js (For pushing apps to the next level)

  • Node.js (For that extra backend support)

  • Figma ( Because Design first before Coding)

  • Sass and Boostrap ( Heard of Atomic CSS ? Better organised CSS )

  • Git & GitHub ( Because your code shouldn't stay within the 4 walls of your HardDisk)
